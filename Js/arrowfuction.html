Arrow function은 function을 대체하는 신문법이 아님

프로그래밍할 때 function 문법은 왜 사용하는지 알아보자)

  • 1. 여러가지 기능을 하는 코드를 한 단어로 묶고 싶을 때 (그리고 나중에 재사용할 때)
  • 2. 입출력기능을 만들 때
  • 3. arrow function을 사용하면 함수 본연의 입출력기능을 직관적으로 잘 표현해 주기 때문에 사용.
  • 4. 함수는 수학에서 온 개념으로 숫자를 넣으면 다른 숫자를 배출해주는 역할을 함
  • 5. 학교에서 배웠던 예시를 들자면 ex ] f(x) = x + 2
  • 6. 5번을 프로그래밍으로 보면? ex ] function plus(x){return x+2}
  • 7. 6번을 Arrow function으로 한줄로 만들 수 있다 ex ] let plus = x => x+2;
